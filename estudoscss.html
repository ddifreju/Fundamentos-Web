1. Introdução ao CSS

O CSS (Cascading Style Sheets) é a linguagem que dá vida e estilo aos seus documentos HTML. Ele define como os elementos da página devem ser apresentados no navegador.

Anatomia de uma Regra CSS

Cada regra CSS é composta por:

  • Seletor: Identifica o(s) elemento(s) HTML a serem estilizados.
  • Bloco de Declaração: Contém uma ou mais declarações.
  • Declaração: Par de propriedade e valor (ex: font-size: 20px;).
p { /* Seletor */
    color: blue; /* Declaração: propriedade: valor; */
    font-size: 16px;
}

Como Adicionar CSS ao HTML

Três formas principais de vincular CSS à sua página:

  • Tag <link> (Recomendado): Vincula um arquivo CSS externo. Ideal para projetos maiores e organização. 
    <link rel="stylesheet" href="style.css">
  • Tag <style>: CSS diretamente dentro do <head> do HTML. Bom para estilos específicos de uma única página. 
    <style>
    h1 { color: green; }
</style>
  • Estilos Embutidos: CSS direto no atributo style de uma tag HTML. Para estilos muito específicos e pontuais. 
    <p style="color: purple;">Texto roxo.</p>

2. Seletores Básicos do CSS

Seletores são a chave para aplicar estilos. Eles "selecionam" os elementos HTML que você deseja estilizar.

Tipos de Seletores

  • Seletor de Elemento: Aplica a todas as instâncias de uma tag HTML. 
    p { /* Seleciona todas as tags <p> */
    line-height: 1.5;
}
  • Seletor de ID (#): Para um único elemento (ID deve ser único). 
    <div id="cabecalho">...</div>
    #cabecalho { /* Seleciona o elemento com id="cabecalho" */
    background-color: #f8f8f8;
}
  • Seletor de Classe (.): Para múltiplos elementos que compartilham a mesma classe. 
    <p class="destaque">...</p>
<span class="destaque">...</span>
    .destaque { /* Seleciona elementos com class="destaque" */
    font-weight: bold;
}
  • Seletor Universal (*): Seleciona todos os elementos HTML. Usado para "resetar" estilos padrão. 
    * { /* Aplica a todos os elementos */
    margin: 0;
    padding: 0;
}

Agrupamento de Seletores

Aplique as mesmas regras a múltiplos seletores separando-os por vírgula.

h1, h2, p { /* Estiliza h1, h2 e p */
    font-family: sans-serif;
    color: #333;
}

Combinações (Exemplos Comuns)

  • Descendente: div p (seleciona <p> dentro de <div>)
  • Filho Direto: ul > li (seleciona <li> que são filhos diretos de <ul>)
  • Irmão Adjacente: h1 + p (seleciona <p> que vem imediatamente após <h1>)
  • Irmão Geral: h1 ~ p (seleciona <p> que são irmãos de <h1> e vêm depois)

3. Unidades de Medida CSS

As unidades de medida são essenciais para criar designs responsivos que se adaptam a diferentes tamanhos de tela.

Unidades Absolutas (Fixas)

  • px (pixel): Unidade fixa, relativa à resolução do dispositivo. 1px = 1/96 de uma polegada. 
    .elemento { width: 150px; }
  • Outras (menos comuns para web): cm, mm, in (polegadas), pt (pontos), pc (picas).

Unidades Relativas (Flexíveis)

  • em: Relativa ao tamanho da fonte do elemento pai mais próximo. 
    .pai { font-size: 16px; }
.filho { font-size: 1.2em; /* 1.2 * 16px = 19.2px */ }
  • rem (Root EM): Relativa ao tamanho da fonte do elemento raiz (). Garante consistência e escalabilidade. 
    html { font-size: 16px; }
h1 { font-size: 2.5rem; /* 2.5 * 16px = 40px */ }
  • Percentual (%): Relativa a uma dimensão do elemento pai. 
    .container { width: 800px; }
.item { width: 50%; /* 50% de 800px = 400px */ }
  • vh (viewport height): 1% da altura da viewport.
  • vw (viewport width): 1% da largura da viewport. 
    .hero { height: 100vh; /* Ocupa 100% da altura da tela */ }
.titulo { font-size: 5vw; /* 5% da largura da tela */ }
  • vmin: 1% da menor dimensão da viewport (altura ou largura).
  • vmax: 1% da maior dimensão da viewport (altura ou largura).

4. Cores no CSS

A escolha de cores vai além da estética; é uma decisão estratégica baseada na Teoria das Cores.

Teoria das Cores Aplicada ao Design

  • Psicologia das Cores: Como cada cor evoca emoções (ex: azul = confiança, vermelho = energia).
  • Equilíbrio: Use uma paleta limitada (3-5 cores). Regra 60-30-10: 60% cor dominante, 30% secundária, 10% de destaque.
  • Acessibilidade: Garanta bom contraste entre texto e fundo para legibilidade, especialmente para pessoas com deficiência visual.

Formas de Implementar Cores no CSS

  • Nomes: Simples e legível para cores básicas. 
    color: red;
background-color: blue;
  • RGB/RGBA: Define cores usando os valores de Vermelho, Verde e Azul (0-255). RGBA adiciona um canal Alfa para transparência (0-1). 
    color: rgb(255, 0, 0); /* Vermelho puro */
background-color: rgba(0, 128, 0, 0.5); /* Verde com 50% de transparência */
  • Hexadecimal (#): O formato mais comum, usando 6 dígitos hexadecimais (0-F) para representar RGB (ex: #RRGGBB). 
    color: #FF0000; /* Vermelho puro */
background-color: #3366FF; /* Um tom de azul */
  • HSL/HSLA: Define cores usando Hue (matiz), Saturation (saturação) e Lightness (luminosidade). HSLA também adiciona o canal Alfa. 
    color: hsl(0, 100%, 50%); /* Vermelho puro */
background-color: hsla(120, 100%, 50%, 0.7); /* Verde com 70% de opacidade */

Propriedades Principais

  • color: Para a cor do texto.
  • background-color: Para a cor de fundo de um elemento.

5. Variáveis CSS

As Variáveis CSS (ou Propriedades Customizadas) permitem armazenar valores reutilizáveis, tornando o código mais limpo, fácil de manter e flexível.

Declaração de Variáveis

Geralmente declaradas no seletor :root para serem acessíveis globalmente. O nome da variável deve começar com dois traços (--). 

:root {
    --cor-principal: #007bff;
    --espacamento-padrao: 16px;
    --fonte-titulo: 'Inter', sans-serif;
}

Uso de Variáveis

Para aplicar o valor de uma variável, utilize a função var(). 

body {
    font-family: var(--fonte-titulo);
    background-color: #f0f4f8;
}

.botao {
    background-color: var(--cor-principal);
    padding: var(--espacamento-padrao);
    color: white;
    border-radius: 8px;
}

Benefícios: Manutenção simplificada (mude um valor em um só lugar), melhor legibilidade e consistência no design.

6. Tipografia no CSS

A tipografia é a arte de estilizar o texto. A escolha e o uso corretos das fontes são cruciais para a legibilidade e a estética do seu site.

Propriedades Essenciais

  • font-family: Define a fonte. Use uma lista de "fallback" para compatibilidade. 
    body {
    font-family: 'Roboto', Arial, sans-serif;
}

    Fontes podem ser "Web Safe" (padrão do sistema) ou importadas (ex: Google Fonts).

  • font-weight: Controla a espessura do texto. 
    h1 { font-weight: bold; } /* ou 700 */
p { font-weight: 400; } /* Normal */
  • font-style: Aplica estilos como itálico. 
    em { font-style: italic; }
  • font-size: Define o tamanho do texto (use unidades relativas como rem ou em para responsividade). 
    p { font-size: 1rem; }
h2 { font-size: 2em; }
  • line-height: Espaçamento entre linhas.
  • text-align: Alinhamento do texto (left, right, center, justify).
  • text-decoration: Adiciona ou remove decorações (ex: underline, none).
  • text-transform: Transforma o texto (uppercase, lowercase, capitalize).

7. O Box Model do CSS

No CSS, todo elemento HTML é tratado como uma caixa retangular. O Box Model descreve a composição dessa caixa em quatro camadas concêntricas.

As Quatro Camadas

  • Conteúdo (Content): A área onde o texto, imagens ou outros elementos aparecem.
  • Padding (Preenchimento): O espaço interno entre o conteúdo e a borda. Adiciona "respiro" ao elemento. 
    .caixa { padding: 20px; }
  • Border (Borda): Uma linha que envolve o padding e o conteúdo. 
    .caixa { border: 2px solid #ccc; }
  • Margin (Margem): O espaço externo, que separa o elemento de outros elementos vizinhos. 
    .caixa { margin: 10px; }

box-sizing: border-box; (Fundamental!)

Por padrão, a largura e altura de um elemento incluem apenas o conteúdo. Com box-sizing: border-box;, o padding e a border são incluídos na largura e altura totais, tornando os layouts muito mais previsíveis e fáceis de trabalhar. 

/* Geralmente aplicado globalmente */
html {
    box-sizing: border-box;
}
*, *::before, *::after {
    box-sizing: inherit;
}

Visualização: Imagine uma cebola, onde cada camada é uma parte do Box Model.

8. CSS Flexbox

O Flexbox (Flexible Box Layout) é um modelo de layout unidimensional (1D). É perfeito para alinhar e distribuir espaço entre itens em uma única linha ou coluna.

Ativação e Eixos

  • Ativação: Aplique display: flex; a um elemento pai (o contêiner flex).
  • Eixos:
    • Main Axis (Eixo Principal): A direção em que os itens flex se organizam (horizontal por padrão).
    • Cross Axis (Eixo Transversal): Perpendicular ao eixo principal.
.container-flex {
    display: flex;
    /* Itens se organizarão em linha por padrão */
}

Principais Propriedades (do Contêiner)

  • flex-direction: Define a direção dos itens (row, column, row-reverse, column-reverse). 
    flex-direction: column;
  • justify-content: Alinha os itens ao longo do eixo principal. 
    justify-content: center; /* Centraliza */
justify-content: space-between; /* Espaço entre itens */
  • align-items: Alinha os itens ao longo do eixo transversal. 
    align-items: center; /* Centraliza verticalmente (se direction: row) */
  • flex-wrap: Permite que os itens quebrem para a próxima linha (wrap) se não houver espaço. 
    flex-wrap: wrap;

9. CSS Grid Layout

O CSS Grid Layout é um modelo bidimensional (2D), ideal para organizar elementos em linhas e colunas simultaneamente, perfeito para layouts de página complexos.

Ativação e Estrutura

  • Ativação: Aplique display: grid; a um elemento pai (o contêiner grid).
  • Definindo a Grade:
    • grid-template-columns: Controla o número e o tamanho das colunas.
    • grid-template-rows: Controla o número e o tamanho das linhas.
    .container-grid {
    display: grid;
    grid-template-columns: 1fr 1fr 1fr; /* Três colunas de tamanhos iguais */
    grid-template-rows: auto 100px; /* Primeira linha auto, segunda com 100px */
}

    A unidade fr (fração) é muito usada para distribuir o espaço disponível.

Espaçamento e Posicionamento

  • gap: Cria espaçamento entre as células da grade. 
    gap: 20px; /* Espaçamento de 20px entre linhas e colunas */
row-gap: 10px; /* Apenas entre linhas */
column-gap: 15px; /* Apenas entre colunas */
  • Posicionamento de Itens: Use grid-column e grid-row para posicionar itens específicos na grade. 
    .item-1 {
    grid-column: 1 / 3; /* Ocupa da coluna 1 até a 3 (ou seja, 2 colunas) */
    grid-row: 1;
}

Grid vs. Flexbox

  • Grid: Para o layout geral da página (2D - linhas E colunas).
  • Flexbox: Melhor para alinhar componentes e conteúdos internos (1D - linha OU coluna).
  • Juntos: Eles se complementam! Use Grid para o layout principal e Flexbox dentro das células do Grid para alinhar conteúdos.
